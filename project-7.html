Techdegree Project -7- Webapp Dashboard

For this project I created all of the HTML, CSS, and JavaScript from scratch.

I used Grid Layout and Flexbox to layout and arrange all the different sections of the web page. I used mobile-first responsive web design.

I used JavaScript to create an alert for tasks. For this alert I used string interpolation to add the HTML to the web page.

I implemented charts using charts.js.

I used JavaScript to import the data for the traffic chart via an object with data and labels.

For all the other chart data and styling customization, I edited the object parameters used by the charts in JavaScript.

I made all charts responsive. I used SVG's to implement the social networking icons in their respective cards.

I edited SVG parameters to change the fill of each social networking icon.

I used margins to fully align SVG's to the center of their containers.

I implemented message sending functionality for the messaging section, where if you type a message into the message field and click send, it will send the data to the user specified and then clear the data from the message field.

For the settings section I created custom toggle switches from a w3schools example.

I also added a timezone dropdown field and a save button to save the settings to the browser using local storage. This project used JavaScript arrays, objects, functions, loops and conditionals.

This project was designed according to the provided mockups.
